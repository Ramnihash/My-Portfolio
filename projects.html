Refine.AI

Refine.AI is an AI-powered tool designed to help users improve and structure their prompts with clarity and intent. It focuses on guiding users through refinement rather than generation, enabling better outputs by adapting prompts based on context, audience, and purpose.

Legend Ecommerce

Stay Sure

NoteIT

NoteIT is a simple web project focused on organizing daily tasks and notes in one place. I worked on structuring the interface, designing a clear task flow, and implementing basic features like task creation, completion states, and simple categorization to improve everyday productivity.
